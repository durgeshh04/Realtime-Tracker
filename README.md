# This is a project for tracking real-time data.
## Technologies Used
- Node
- Express
- Socket.io

## Features
- Real-time data tracking
- Data visualization

## Installation
1. Clone the repository
2. Install dependencies
3. Start the server

## Usage
1. Open the client in your browser
2. Log in or register
3. Start tracking data
